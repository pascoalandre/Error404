Guia de Instalação

Siga estes passos para colocar a sua loja online em minutos.

Arquitetura

Conexão Admin-Backend e Função do SQL

Entenda como o Painel Admin interage com a base de dados e a importância do ficheiro `appfystore.sql`.

1. Conexão do Painel Admin com o Banco de Dados

O Painel Admin (frontend) não se conecta diretamente à base de dados. Ele se comunica com o Backend PHP (API), que por sua vez interage com o MySQL de forma segura.

  • Painel Admin (Frontend): Envia requisições HTTP (AJAX/Fetch) para o Backend PHP.
  • Backend PHP (API): Recebe requisições, realiza autenticação, lógica de negócio e interage com o MySQL usando Prepared Statements.
  • Base de Dados MySQL: Armazena dados de apps, categorias, utilizadores admin.
  • Resposta: Backend PHP retorna JSON para o Painel Admin.

Este modelo garante segurança, mantendo credenciais e lógica sensível no servidor.

2. Propósito do Ficheiro `appfystore.sql`

O `appfystore.sql` é um script SQL essencial para configurar a base de dados do zero. Ele serve para:

  • Criação do Esquema: Define a estrutura de todas as tabelas (`apps`, `categories`, `admin_users`, etc.), colunas, chaves e índices.
  • População de Dados Iniciais: Insere dados essenciais como o utilizador admin padrão e categorias iniciais, deixando a base de dados pronta para uso.
